Getting Started

This is HTML template. To getting started you need only Browser to run HTML files.

To customize template you can edit CSS and HTML files. Also if you are developer, there are available source files SCSS and JS (required ES6 transpiller). All used plugins added from Npm.

For each pages available blank CSS file for your customizations. See it here
